### 安装
```powershell
# 方便更新
> scoop install idea-ultimate

# 用来设置右键菜单上下文,可以直接把一个文件夹当成idea项目打开
> scoop install hibit-uninstaller
# 这个也可以安装看看
> scoop install jetbrains-toolbox
```

Open Folder as IntelliJ IDEA Project    -   文件夹
Open File as IntelliJ IDEA Project      -   文件
